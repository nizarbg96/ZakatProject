What is Zakat-Reminder Application

First the application will allow the user to register and have his secure space after activation via e-mail, the user will authenticate and thus link his bank account to the application if the bank dedicated to his account already exhibits a secure open API rest so he can get all his balances history to the application database, otherwise the user will enter his balances manually each month.

Following these scales, an algorithm will calculate the periods due for the user where he has to pay Zakat, when the user got a period for which he have to pay Zakat, the user will receive a notification in his mobile phone that reminds him to pay the mandatory alms , therefore he can pay this Zakat and consult after the platform to record this payment, at any time the user can see the history of his payments and the amount remaining to be paid for the periods due to him. the Zakat-Reminder application Provides the user with all the statistics, graphs and tables of deposited scales, payments and all informations entered by the him so that he can track his activity of the Zakat process. The user can view, modify, delete and manage all datas related to his profile.
